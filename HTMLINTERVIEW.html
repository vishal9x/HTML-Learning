1. What is HTML?
Answer: HTML (HyperText Markup Language) is the standard markup language used to create webpages and structure content on the internet. It is not a programming language.

Defines elements like

  • Headings
  • Paragraphs
  • Images
  • Forms
  • Links
  • Tables
<h1>This is heading</h1>
<p>This is paragraph</p>

25. Why use alt in images?
Answer: alt text helps screen readers, improves accessibility, and appears when images fail.

26. What happens when webpage loads in browser?
Answer: Browser follows a rendering pipeline from request to final painted UI.
  1. Browser requests webpage
  2. Server sends HTML
  3. Browser parses HTML
  4. CSS applied
  5. JavaScript executed
  6. Page rendered

Important Tip for Interviews

Interviewers usually check semantic understanding, practical coding, accessibility awareness, clean structure, and confidence while explaining.

Frontend interviews often combine HTML, CSS, JavaScript, responsive design, and browser behavior.

43. Difference between script in head and body?
Answer: Scripts in head can block rendering unless async/defer is used. End-of-body scripts usually allow faster initial render.

44. What is accessibility in HTML?
Answer: Accessibility means making websites usable for disabled users, screen readers, and keyboard navigation.

Examples include semantic tags, alt text, and form labels.
